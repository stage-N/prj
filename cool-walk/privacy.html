プライバシーポリシー

STAGEN（以下「当社」）は、Cool Walk（以下「本アプリ」）における個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリは、WBGT・UV・大気質データを取得する目的で、端末の位置情報を利用します。

  • 位置情報は天気・大気質データの取得のみに使用されます
  • 位置座標は Open-Meteo API へのリクエスト時にのみ送信されます
  • ユーザー登録は不要です
  • HealthKit 等の健康データは MVP 版では使用しません

2. 端末内に保存されるデータ

以下のデータは端末内にのみ保存され、当社サーバーには送信されません：

  • 活動種別・通知設定
  • 最後に取得した WBGT・UV・PM 予報データ（オフライン表示用）

3. 外部通信

  • Open-Meteo Forecast API — WBGT 推定用の天気データ取得
  • Open-Meteo Air Quality API — PM2.5 / PM10 データ取得
  • OpenStreetMap Overpass API — 公園・緑地データ取得（日陰ルート推薦）
  • OSRM — 徒歩ルート計算（日陰ルート推薦）
  • 国土地理院タイル — 地図表示

いずれも位置座標（緯度・経度）のみを送信します。

4. 通知について

本アプリはローカルプッシュ通知を提供します。毎朝7時のおすすめ時間帯通知、WBGT 28°C以上の警告通知は端末内でスケジュールされます。

5. 広告・アナリティクス

本アプリは広告表示およびアクセス解析ツールを使用していません。

6. 第三者への提供

ユーザーデータの第三者提供は行いません。

7. データの削除

本アプリをアンインストールすることで、端末内のすべてのデータが削除されます。

8. 免責事項

本アプリは医療機器ではありません。提供する情報は外出の目安であり、医療診断・治療の助言ではありません。熱中症予防や健康に関する判断は、必ず医師等の専門家にご相談のうえ行ってください。

9. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。
