プライバシーポリシー

STAGEN（以下「当社」）は、ZaiTap（ザイタップ）および ZaiTap 行政書士（以下あわせて「本サービス」）における個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 本サービスの構成

本サービスは次の2つのアプリで構成されます。

  • ZaiTap（申請者） — 在留外国人本人向け。在留手続の入力・提出・進捗確認・行政書士への相談申込
  • ZaiTap 行政書士 — 提携行政書士向け B2B ツール。相談依頼の受信・書類閲覧・連絡

2. 収集する情報

本サービス利用にあたり、以下の情報を処理する場合があります。

  • 端末識別子（セッション管理・Push 通知用）
  • 在留手続に関する入力内容（申請 XML、相談 summary 等）
  • 暗号化された添付書類（相談時にアップロードされた場合）
  • 費用目安への同意記録（タイムスタンプ・手続種別・金額レンジ）
  • 行政書士アプリ：事務所名、端末 Store ID、サブスクリプション状態

本人申請機能の基本利用にアカウント登録は不要です（行政書士アプリの登録を除く）。

3. 情報の利用目的

  • 在留申請手続 API への提出代行（本人申請）
  • 行政書士との相談・連絡の仲介
  • Push 通知の送信
  • 費用目安表示・同意ログの記録
  • 行政書士向けサブスクリプション・マッチング枠の管理

4. サーバー・保存

相談・連絡機能に限り、Cloudflare Workers（D1 データベース・R2 ストレージ）上でデータを処理します。暗号化書類は R2 に保存され、一定期間後に自動削除されます。申請 XML のローカル生成・NFC 署名は端末内で処理されます。

5. 第三者への提供

  • 出入国在留管理庁 — 本人申請時、API 経由で申請データを送信
  • 提携行政書士 — 相談申込・受諾後、依頼人が同意した範囲の相談情報・暗号化書類
  • Expo Push — 通知配信（トークンのみ）
  • RevenueCat / App Store / Google Play — 行政書士アプリのサブスクリプション・課金処理のみ

申請者アプリに広告 SDK は使用しません。申請者向けアプリ内課金は行いません。

6. 行政書士法に関する注記

申請者アプリのダウンロード・提出機能は無料です。当社は申請者から手数料を徴収しません。行政書士への報酬は依頼人と行政書士の直接契約に基づきます。当社は紹介・連絡・IT プラットフォーム提供にとどまります。

7. お子様のプライバシー

本サービスは13歳未満を対象としていません。

8. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点で効力を生じます。

9. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。
