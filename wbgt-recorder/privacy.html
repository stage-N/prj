プライバシーポリシー

STAGEN（以下「当社」）は、熱中症レコーダー Pro（以下「本アプリ」）における個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリは、WBGT（暑さ指数）を取得する目的で、端末の位置情報を利用します。

  • 位置情報はWBGT値の取得のみに使用されます
  • 位置座標は Open-Meteo API への天気データリクエスト時にのみ送信されます
  • ユーザー登録や広告目的の個人情報収集は行いません

2. 端末内に保存されるデータ

以下のデータは SQLite データベースとして端末内にのみ保存され、当社サーバーには送信されません：

  • 作業記録（開始・終了時刻、作業種別、作業者数など）
  • 会社名・現場名などの設定
  • 通知・閾値の設定
  • WBGT値と記録の紐づけデータ

外部サーバーへのデータ同期・バックアップは行いません。アプリを削除すると、保存データもすべて削除されます。

3. 外部通信

本アプリは以下の外部サービスと通信を行います。

  • Open-Meteo API — 天気データ取得のため、位置座標（緯度・経度）を送信
  • 環境省 暑さ指数サイト — 暑さ指数の参照データを取得

上記以外の外部サービスへの通信は、ライトプラン（アプリ内課金）の購入・復元時を除き行いません。

4. ライトプラン（アプリ内課金）について

ライトプランの購入・復元時のみ、RevenueCat および App Store / Google Play を経由して課金処理が行われます。この際に送信されるのは、ストアが課金処理のために必要とする匿名の購入情報のみです。当社がユーザーの作業記録やWBGTデータを RevenueCat に送信することはありません。

5. 通知について

本アプリはローカルプッシュ通知を提供します。通知は端末内でスケジュールされ、外部サーバーは関与しません。

6. アナリティクス・広告

本アプリは、アクセス解析ツール（アナリティクス）および広告SDKを一切使用していません。

7. 第三者への提供

作業記録やWBGTデータの第三者提供は行いません。課金処理に限り、Apple / Google / RevenueCat の各サービス提供者に課金に必要な情報が渡されます。

8. ポリシーの変更

本ポリシーを改定する場合は、GitHub上で事前に告知いたします。

9. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。
