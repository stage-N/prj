개인정보 처리방침

STAGEN(이하 "당사")은 Cool Walk(이하 "본 앱")에서의 개인정보 처리에 관하여 다음과 같이 개인정보 처리방침을 정합니다.

1. 수집하는 정보

본 앱은 WBGT·UV·대기질 데이터를 취득하기 위해 기기의 위치 정보를 이용합니다.

  • 위치 정보는 날씨·대기질 데이터 취득에만 사용됩니다
  • 위치 좌표는 Open-Meteo API에 대한 요청 시에만 전송됩니다
  • 사용자 등록은 필요하지 않습니다
  • HealthKit 등 건강 데이터는 MVP 버전에서 사용하지 않습니다

2. 기기 내 저장 데이터

다음 데이터는 기기 내에만 저장되며 당사 서버로 전송되지 않습니다:

  • 활동 종류·알림 설정
  • 마지막으로 취득한 WBGT·UV·PM 예보 데이터(오프라인 표시용)

3. 외부 통신

  • Open-Meteo Forecast API — WBGT 추정용 날씨 데이터 취득
  • Open-Meteo Air Quality API — PM2.5 / PM10 데이터 취득
  • OpenStreetMap Overpass API — 공원·녹지 데이터 취득(그늘 루트 추천)
  • OSRM — 도보 루트 계산(그늘 루트 추천)
  • 국토지리원 타일 — 지도 표시

어느 경우든 위치 좌표(위도·경도)만 전송합니다.

4. 알림

본 앱은 로컬 푸시 알림을 제공합니다. 매일 오전 7시 추천 시간대 알림, WBGT 28°C 이상 경고 알림은 기기 내에서 예약됩니다.

5. 광고·분석

본 앱은 광고 표시 및 접속 분석 도구를 사용하지 않습니다.

6. 제3자 제공

사용자 데이터의 제3자 제공은 하지 않습니다.

7. 데이터 삭제

본 앱을 제거하면 기기 내의 모든 데이터가 삭제됩니다.

8. 면책 사항

본 앱은 의료기기가 아닙니다. 제공하는 정보는 외출의 참고이며 의료 진단·치료 조언이 아닙니다. 온열질환 예방 및 건강에 관한 판단은 반드시 의사 등 전문가와 상담한 뒤 해 주세요.

9. 문의

본 방침에 관한 문의는 지원 페이지를 통해 연락해 주세요.
