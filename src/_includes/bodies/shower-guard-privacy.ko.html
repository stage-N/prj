개인정보 처리방침

STAGEN(이하 "당사")은 Shower Guard(이하 "본 앱")에서의 개인정보 처리에 관하여 다음과 같이 개인정보 처리방침을 정합니다.

1. 수집하는 정보

본 앱은 강수 예보를 취득하기 위해 기기의 위치 정보를 이용합니다.

  • 위치 정보는 강수 예보 취득에만 사용됩니다
  • 위치 좌표는 Open-Meteo API에 대한 날씨 데이터 요청 시에만 전송됩니다
  • 사용자 등록은 필요하지 않습니다
  • 계정·로그인 기능은 없습니다

2. 기기 내 저장 데이터

다음 데이터는 기기 내에만 저장되며 당사 서버로 전송되지 않습니다:

  • 알림 설정(출발 시각, 감도)
  • 마지막으로 취득한 강수 예보 데이터(오프라인 표시용)
  • 위젯 표시용 텍스트

3. 외부 통신

  • Open-Meteo Weather API — 강수 예보 취득을 위해 위치 좌표(위도·경도) 전송

4. 알림

본 앱은 로컬 푸시 알림을 제공합니다. 알림은 기기 내에서 예약되며 당사 서버는 관여하지 않습니다. APNs(Apple Push Notification service) 이용 자격은 보유하고 있으나 MVP 버전에서는 서버에서의 원격 푸시는 하지 않습니다.

5. 광고·분석

본 앱은 광고 표시 및 접속 분석 도구를 사용하지 않습니다.

6. 제3자 제공

사용자 데이터의 제3자 제공은 하지 않습니다.

7. 데이터 삭제

본 앱을 제거하면 기기 내의 모든 데이터가 삭제됩니다.

8. 면책 사항

본 앱의 강수 예보는 참고 정보이며 정확성을 보장하지 않습니다. 외출의 최종 판단은 본인의 책임으로 해 주세요.

9. 문의

본 방침에 관한 문의는 지원 페이지를 통해 연락해 주세요.
