#작업현장 #온열질환대책 #산업안전보건법

온열질환 레코더 Pro현장의 더위 대책

온열지수(WBGT)를 실시간으로 측정·기록합니다. 작업 시작부터 종료까지 원탭으로 기록하고 CSV·PDF로 보냅니다. 현장 안전 관리를 스마트폰 하나로.

계정 불필요 기기 내 저장 iOS / Android
거의 안전 주의 경계 엄중 경계 위험

WBGT 기록이 필요한 이유

환경성 기준 온열지수로 온열질환 위험을 수치화합니다.

산업안전보건법 대응

옥외·실내 고온 작업에서는 WBGT 값의 파악과 기록이 안전 관리의 기본입니다. 본 앱은 작업 기록과 WBGT 값을 연결해 저장하며, 3년간의 기록 관리에 대응합니다.

종이 일보에서 스마트폰으로

반원 게이지로 현재 WBGT를 한눈에 확인합니다. 24시간 예보로 작업 계획을 세우고, 임계값을 초과하면 푸시 알림. 현장 감독의 부담을 줄이고 놓침을 방지합니다.

주요 기능

작업 현장의 더위 대책에 필요한 기능을 하나로.

📊

WBGT 실시간 표시

Open-Meteo와 환경성 온열지수 데이터를 통합합니다. 환경성 데이터가 이용 가능하면 우선 표시합니다.

⏱️

작업 기록

시작·일시정지·재개·종료를 원탭. WBGT 값과 작업 시간을 자동으로 연결해 저장합니다.

👷

작업자 수·현장 관리

작업자 수 입력, 회사명·현장명 설정. 옥외 작업·건설 작업 등 7종류의 작업 프리셋.

📄

CSV / PDF 보내기

기록을 일보·월보로 출력합니다. 현장 보고·보관 업무를 효율화합니다.

🚨

위험 수준 알림

설정한 임계값을 초과하면 로컬 푸시 알림. 백그라운드에서도 WBGT를 모니터링합니다.

📈

24시간 예보

향후 24시간의 WBGT 추이를 그래프로 확인합니다. 작업 시작 시각 판단에 활용할 수 있습니다.

앱 실제 화면

직관적인 UI로 현장에서도 바로 사용할 수 있습니다.
