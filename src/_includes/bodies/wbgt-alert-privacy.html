プライバシーポリシー

STAGEN（以下「当社」）は、熱中症アラート（以下「本アプリ」）における個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリは、WBGT（暑さ指数）を取得する目的で、端末の位置情報を利用します。

  • 位置情報はWBGT値の取得のみに使用されます
  • 位置座標は Open-Meteo API への天気データリクエスト時にのみ送信されます
  • ユーザー登録は不要です

2. 端末内に保存されるデータ

以下のデータは SQLite データベースとして端末内にのみ保存され、当社サーバーには送信されません：

  • 活動記録（開始・終了時刻、活動種別など）
  • 個人プロフィール（年齢層・既往、任意設定）
  • 通知・閾値の設定
  • WBGT値と記録の紐づけデータ

個人プロフィールに基づく閾値補正は医療アドバイスではありません。アプリを削除すると、保存データもすべて削除されます。

3. 外部通信

  • Open-Meteo API — 天気データ取得のため、位置座標（緯度・経度）を送信
  • 環境省 暑さ指数サイト — 暑さ指数の参照データを取得

4. 広告について

本アプリは、アプリ起動時およびバックグラウンドからの復帰時に、Google AdMob の App Open 広告を表示します。広告の配信にあたり、Google がデバイス識別子や広告 ID などの情報を収集する場合があります。

  • 利用する広告 SDK: Google AdMob
  • iOS では、より関連性の高い広告のためにトラッキング許可をお願いする場合があります（拒否してもアプリの利用や広告表示には影響しません）
  • 広告のパーソナライズを制限するには、端末の広告設定をご利用ください
  • 熱中症警告表示中は広告を表示しません

5. 通知について

本アプリはローカルプッシュ通知を提供します。通知は端末内でスケジュールされ、外部サーバーは関与しません。

6. アナリティクス

本アプリは、アクセス解析ツール（アナリティクス）を使用していません。

7. 第三者への提供

活動記録やWBGTデータの第三者提供は行いません。広告配信に限り、Google に広告表示に必要な情報が渡される場合があります。

8. ポリシーの変更

本ポリシーを改定する場合は、GitHub上で事前に告知いたします。

9. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。
