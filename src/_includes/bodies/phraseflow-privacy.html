プライバシーポリシー

株式会社 STAGEN（以下「当社」）は、Phrase Flow（フレーズフロー、以下「本サービス」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。

無料機能はアカウント登録なしで利用できます。Premium 購読時のみ、決済に必要な情報を Stripe 経由で処理します。

1. 収集する情報

  • 無料利用時: 名言の閲覧に氏名・メール等は不要です（行動追跡用の広告 SDK は使用しません）
  • アカウント登録時: メールアドレスとパスワード（ハッシュ化して保存）を Cloudflare D1 に保存します
  • Premium 購読時: Stripe 決済に必要なメールアドレス等を Stripe が処理します。当社は購読状態の管理に必要な識別子をサーバーに保存します
  • メール配信（オプトイン）: 配信先メールアドレスと配信設定を保存します
  • アクセスログ: Cloudflare のインフラログ（IP・リクエスト時刻等）が技術的に記録される場合があります

2. AI処理・名言データ

本サービスは、キュレーションされた名言シードを Google Gemini API で背景・解説・対照意見に展開し、多言語テンプレートを生成します。生成されたコンテンツは表示・配信のために当社サーバー（Cloudflare D1）に保存されます。

ユーザーの端末から Gemini へ個人を特定できるデータが直接送信されることはありません（AI処理は当社バックエンドが行います）。

3. 第三者サービス

  • Stripe: 有料プランの決済・サブスクリプション管理
  • Google Gemini API: 名言の背景・解説・対照意見の生成
  • Cloudflare: ホスティング・データベース（D1）・セッション（KV）
  • RevenueCat（アプリ）: モバイル購読状態の同期（予定）

4. 第三者への提供

法令に基づく場合を除き、ユーザーの個人情報を第三者に販売・提供しません。

5. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。

6. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点で効力を生じます。
