プライバシーポリシー

株式会社 STAGEN（以下「当社」）は、Touten（トウテン、以下「本サービス」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。

無料機能はアカウント登録なしで利用できます。Premium 購読時のみ、決済に必要な情報を Stripe 経由で処理します。

1. 収集する情報

  • 無料利用時: 氏名・メール等の個人を特定できる情報は収集しません（Cookie による行動追跡・広告 SDK は使用しません）
  • Premium 購読時: Stripe 決済に必要なメールアドレス等を Stripe が処理します。当社は購読状態の管理に必要な識別子をサーバー（Cloudflare D1）に保存します
  • アクセスログ: Cloudflare のインフラログ（IP・リクエスト時刻等）が技術的に記録される場合があります

2. 外部データ取得・AI処理

本サービスは、日本5大全国紙の公開 RSS 等から見出し・本文を取得し、Google Gemini API で要約・スタンス分析を行います。取得した記事内容は分析・表示のために当社サーバーに保存されます。

ユーザーの端末から、購読していない第三者メディアへ個人データが送信されることはありません（記事取得は当社バックエンドが行います）。

3. 第三者サービス

  • Stripe: 有料プランの決済・サブスクリプション管理
  • Google Gemini API: 記事の要約・スタンス分析
  • Cloudflare: ホスティング・データベース（D1）・キャッシュ（KV）

4. 第三者への提供

法令に基づく場合を除き、ユーザーの個人情報を第三者に販売・提供しません。

5. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。

6. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点で効力を生じます。
