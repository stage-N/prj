プライバシーポリシー

STAGEN（以下「当社」）は、Shower Guard（以下「本アプリ」）における個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリは、降水予報を取得する目的で、端末の位置情報を利用します。

  • 位置情報は降水予報の取得のみに使用されます
  • 位置座標は Open-Meteo API への天気データリクエスト時にのみ送信されます
  • ユーザー登録は不要です
  • アカウント・ログイン機能はありません

2. 端末内に保存されるデータ

以下のデータは端末内にのみ保存され、当社サーバーには送信されません：

  • 通知設定（出発時刻、感度）
  • 最後に取得した降水予報データ（オフライン表示用）
  • ウィジェット表示用のテキスト

3. 外部通信

  • Open-Meteo Weather API — 降水予報取得のため、位置座標（緯度・経度）を送信

4. 通知について

本アプリはローカルプッシュ通知を提供します。通知は端末内でスケジュールされ、当社サーバーは関与しません。APNs（Apple Push Notification service）の利用資格は保持していますが、MVP版ではサーバーからのリモートプッシュは行いません。

5. 広告・アナリティクス

本アプリは広告表示およびアクセス解析ツールを使用していません。

6. 第三者への提供

ユーザーデータの第三者提供は行いません。

7. データの削除

本アプリをアンインストールすることで、端末内のすべてのデータが削除されます。

8. 免責事項

本アプリの降水予報は参考情報であり、正確性を保証するものではありません。出行の最終判断はご自身の責任で行ってください。

9. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。
