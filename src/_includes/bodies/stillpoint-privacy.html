プライバシーポリシー

株式会社 STAGEN（以下「当社」）は、Stillpoint（スティルポイント、以下「本サービス」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。

本サービスは内省の補助であり、医療・心理療法ではありません。危機的な状況では地域の相談窓口・緊急サービスをご利用ください。

1. 収集する情報

  • アカウント登録時: メールアドレスとパスワード（ハッシュ化して保存）を Cloudflare D1 に保存します
  • セッション・内省データ: 考えごと、AI質問、音声から変換されたテキスト内省、ターン時間などを、同期のためサーバーに保存します
  • Premium 購読時: Stripe / App Store / Google Play および RevenueCat 経由で購読状態を同期します。当社は購読識別に必要な ID を保存します
  • アクセスログ: Cloudflare のインフラログ（IP・リクエスト時刻等）が技術的に記録される場合があります

2. AI処理

内省テキストは、より深いソクラテス式の質問を生成するため、当社バックエンド経由で Google Gemini API および（フォールバック時）Cloudflare Workers AI に送信される場合があります。生成結果はセッションの一部として保存されます。

音声認識は原則として端末上またはブラウザの Web Speech API で行い、当社が音声生データを第三者 STT に送ることはありません（実装・OS により異なります）。

3. データの保持（Free）

Free プランのアーカイブは直近7日分です。期限を過ぎたセッションは表示されないだけでなく、サーバー上から削除されます。Premium は無制限に保持します。

4. 第三者サービス

  • Stripe / RevenueCat / App Store / Google Play: 決済・購読
  • Google Gemini API / Cloudflare Workers AI: 深掘り質問の生成
  • Cloudflare: ホスティング・D1・KV

5. 第三者への提供

法令に基づく場合を除き、ユーザーの個人情報を第三者に販売・提供しません。

6. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。

7. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点で効力を生じます。
