プライバシーポリシー

株式会社 STAGEN（以下「当社」）は、PagePace（ページペース、以下「本サービス」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。

本サービスは読書習慣の記録ツールです。第三者の書籍メタデータ（ISBN・表紙等）を取得する場合があります。

1. 収集する情報

  • アカウント登録時: メールアドレスとパスワード（ハッシュ化して保存）を Cloudflare D1 に保存します
  • 読書データ: 書庫（ISBN・タイトル・ページ進捗）、セッション時間、区間ペースなどを、同期のためサーバーに保存します
  • 端末ローカル通知: 毎日の読書リマインダーは端末上でスケジュールされ、当社サーバーへプッシュ配信しません
  • Premium 購読時: Stripe / App Store / Google Play および RevenueCat 経由で購読状態を同期します。当社は購読識別に必要な ID を保存します
  • アクセスログ: Cloudflare のインフラログ（IP・リクエスト時刻等）が技術的に記録される場合があります

2. 書籍メタデータ

ISBN 検索・表紙取得のため、openBD、楽天ブックス、Open Library、Google Books 等の第三者 API を利用する場合があります。入力した ISBN がこれらのサービスに送信されることがあります。

3. データの保持（Free）

Free プランの記録は直近30日分が主対象です。Premium は無制限に保持します。アカウント削除をご希望の場合はサポートまでご連絡ください。

4. 第三者サービス

  • Stripe / RevenueCat / App Store / Google Play: 決済・購読
  • openBD / 楽天 / Open Library / Google Books: 書籍メタデータ・表紙
  • Cloudflare: ホスティング・D1・KV

5. 第三者への提供

法令に基づく場合を除き、ユーザーの個人情報を第三者に販売・提供しません。

6. お問い合わせ

本ポリシーに関するお問い合わせはサポートページよりご連絡ください。

7. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点で効力を生じます。
