핵심 가치

4가지 원칙으로 함수형 사고를 체득

⚡

No setup, just code

설치 불필요. 앱 내장 인터프리터로 즉시 실행. 열자마자 REPL.

💡

Think, not memorize

구문 암기가 아니라 계산 사고(분해·추상화·재귀). AI 시대에 더욱 중요합니다.

λ

Functional first

상태 없는 순수 함수, 불변 데이터, 합성부터 시작. 명령형 복잡성을 피합니다.

🎮

Playful progression

게임처럼 단계를 밟으며 자연스럽게 「함수형의 눈」을 갖게 됩니다.

Fuzen의 구조

Lisp 스타일 미니 언어 + 함수형 커리큘럼 + 모바일 REPL

Phase 0Playground
Phase 1Expression 사고
Phase 2Named Expression
Phase 3리스트와 재귀
Phase 4고차 함수
Phase 5합성과 파이프
🌐

Unicode 완전 지원

일본어·한국어·영어·중국어·태국어. 함수명·변수명에 CJK를 그대로 쓸 수 있습니다.

🔍

Step trace

Run 시 평가 단계를 깊이와 함께 표시. 무엇이 어떤 순서로 평가됐는지 한눈에.
